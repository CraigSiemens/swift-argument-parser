init(wrappedValue:name:parsing:completion:help:)

Creates a property with a default value provided by standard Swift default value syntax.

Declaration

init(wrappedValue: Value, name: NameSpecification = .long, parsing parsingStrategy: SingleValueParsingStrategy = .next, completion: CompletionKind? = nil, help: ArgumentHelp? = nil)

Parameters

A specification for what names are allowed for this flag.

Discussion

This method is called to initialize an Option with a default value such as:

@Option var foo: String = "bar"
